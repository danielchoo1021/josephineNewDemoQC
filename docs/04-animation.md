# Le_Almmora Animation Guidelines

## Purpose

Animation should improve the premium feeling of the website.

Animation should never become the main attraction.

Users should feel smoothness, elegance and quality.

---

# Overall Principles

The animation style should be:

- Elegant
- Smooth
- Slow
- Natural
- Premium

Avoid:

- Bounce
- Flash
- Spin
- Shake
- Gaming effects
- Excessive movement

---

# Hero

When the homepage loads:

Sequence:

1. Header
2. Hero Heading
3. Hero Description
4. CTA Buttons
5. Hero Product
6. Scroll Indicator

Animation:

Fade Up

Duration:

600ms – 900ms

Delay between elements:

80ms – 150ms

---

# Hero Background

Allowed:

- Slow zoom (maximum 1.03 scale)
- Gentle parallax
- Soft gradient movement

Not allowed:

- Fast zoom
- Infinite movement
- Rotating background
- Heavy blur animation

---

# Buttons

Hover:

- Background colour transition
- Border transition
- Slight lift (2px maximum)

Duration:

200ms

Do not:

- Bounce
- Rotate
- Large scaling

---

# Cards

Product cards

Hover:

- Image zoom (1.03)
- Soft shadow increase
- Slight upward movement

Duration:

250ms

---

# Images

Reveal animation:

Fade Up

or

Fade In

Do not use:

- Flip
- Rotate
- Bounce

---

# Scroll Reveal

Each section should appear naturally.

Recommended:

Opacity:

0 → 1

Translate Y:

20px → 0

Duration:

600ms

---

# Navigation

Transparent while Hero is visible.

After scrolling:

- White background
- Small shadow
- Logo slightly smaller

Transition:

300ms

---

# Mobile

Reduce animation.

Remove:

- Heavy parallax
- Large reveal effects

Keep only:

- Fade
- Opacity
- Small translateY

---

# Accessibility

Support:

prefers-reduced-motion

When enabled:

Disable:

- Parallax
- Zoom
- Scroll reveal

Keep only:

Opacity transition if necessary.

---

# Performance

Animation must remain GPU friendly.

Prefer:

transform

opacity

Avoid:

width animation

height animation

left/right animation

filter animation

large blur animation
